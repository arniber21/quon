## A Bell pair, start to finish

Quon's linear type checker ensures every qubit is consumed exactly once. This
program prepares two entangled qubits, measures both, and never lets a qubit
escape unused:

```kotlin
fn bell_state(): Circuit<2, 2, 2, Clifford> = circuit {
    H @0 |> CNOT @(0, 1)
}

fn main(): Q<(Bit, Bit)> = run {
    (q0, q1) <- bell_state() @ qreg(2)
    b0       <- measure(q0)
    b1       <- measure(q1)
    return (b0, b1)
}
```

`quonc` compiles this straight to OpenQASM 3 for Qiskit Aer or real hardware —
walk through it yourself in the [quickstart](/getting-started/quickstart/).
