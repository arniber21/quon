This cookbook's eight pages are a fixed, hand-curated tour of the compiler's
own end-to-end fixtures. The repository also carries a growing
**[`samples/`](https://github.com/arniber21/quon/tree/main/samples) corpus** —
narrative demos organized by taxonomy (learning, algorithms, workflows,
visualization, applications, research, and neutral-atom pedagogy) and
indexed by a machine-readable catalog, [ADR-0025](https://github.com/arniber21/quon/blob/main/docs/adr/0025-samples-corpus-layout.md).

Start with [`samples/README.md`](https://github.com/arniber21/quon/blob/main/samples/README.md)
for the taxonomy and pack-ownership table, or browse the catalog directly:

- [`samples/catalog.yaml`](https://github.com/arniber21/quon/blob/main/samples/catalog.yaml) — every sample's id, path, tags, difficulty, `quonc` args, artifacts, and CI mode
- [`samples/CONTRIBUTING.md`](https://github.com/arniber21/quon/blob/main/samples/CONTRIBUTING.md) — how to add a sample and register it in the catalog

## CI

Every entry's `path` is checked to exist, the schema is validated, and every
`ci: smoke` entry is compiled with the debug `quonc` binary — see
[`quonc/tests/samples_catalog.rs`](https://github.com/arniber21/quon/blob/main/quonc/tests/samples_catalog.rs),
run as part of `just ci-rust`'s workspace test suite (or standalone via
`just ci-samples`).
